# news-m25 项目规划

## 项目概述
- **项目名称**: news-m25
- **包名**: app.news_m25
- **类型**: 仿今日头条新闻阅读应用 (单机版)
- **版本**: 0.1.4

## 技术栈
- **语言**: Kotlin 2.3.20
- **构建工具**: Gradle 9.4.1, AGP 9.1.0
- **UI框架**: Jetpack Compose + Material 3
- **架构**: MVVM + Clean Architecture
- **依赖注入**: Hilt
- **数据库**: Room
- **键值存储**: DataStore
- **图片加载**: Coil
- **目标SDK**: Android 16 (API 36)
- **JDK**: 25
