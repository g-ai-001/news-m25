# news-m25

仿今日头条新闻阅读应用 (单机版)

## 功能特性

- 新闻分类浏览 (推荐、社会、娱乐、体育、科技等)
- 新闻搜索功能
- 新闻详情查看
- 新闻分享功能
- 收藏功能
- 历史记录功能
- 深色模式支持
- 响应式布局 (支持手机、平板、折叠屏)
- 离线缓存管理 (自动清理过期缓存)
- 清除缓存功能
- 文字大小调整 (小/中/大/特大)
- 阅读统计 (阅读篇数、阅读时长)
- 新闻排序 (最新/最早/最热)
- 视频播放功能 (支持播放控制、全屏播放、视频缩略图)
- 个性化推荐 (基于阅读历史智能推荐)
- 阅读位置记忆 (自动恢复上次阅读进度)
- 分类管理 (自定义显示/隐藏新闻分类)
- 稍后阅读功能 (书签式待读列表)
- 新闻评分 (1-5星评级)
- 语音播报 (TTS朗读新闻内容)

## 技术栈

- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Room 数据库
- Hilt 依赖注入
- Navigation Compose
