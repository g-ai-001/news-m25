# news-m25

仿今日头条新闻阅读应用 (单机版)

## 功能特性

- 新闻分类浏览 (推荐、社会、娱乐、体育、科技等)
- 新闻搜索功能
- 新闻详情查看
- 新闻分享功能
- 收藏功能
- 历史记录功能
- 深色模式支持
- 响应式布局 (支持手机、平板、折叠屏)
- 清除缓存功能

## 技术栈

- Kotlin 2.3.20
- Jetpack Compose + Material 3
- MVVM + Clean Architecture
- Room 数据库
- Hilt 依赖注入
- Navigation Compose
